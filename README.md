# NYC 311 Service Requests Analysis

## How to Run

1. Make sure you have Python 3 installed.
2. Navigate to this folder in your terminal.
3. Run the script:

python3 analysis.py

Output will be saved to `output.txt`. The console will confirm when the file has been written.

## What This Script Does

This script reads a CSV file of NYC 311 service requests from January through April 2024 and generates a summary report. It counts how many requests are currently open, identifies the most common complaint type, and breaks down the total number of requests by borough. The results are written to `output.txt`.

## Dependencies

This script uses only Python's built-in libraries: `csv`.
